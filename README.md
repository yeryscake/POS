# POS Yery'sCake — Cómo continuar el desarrollo con Claude Code

## Pasos para empezar

1. Descarga y descomprime este ZIP en una carpeta de tu computadora,
   por ejemplo: `Documentos/pos-yeryscake`
2. Instala Claude Code (app de escritorio de Claude → pestaña Code).
3. Abre Claude Code y selecciona la carpeta del proyecto.
4. Claude Code leerá automáticamente el archivo CLAUDE.md con todo el
   contexto de lo que ya construimos.
5. Primer mensaje sugerido:
   "Lee CLAUDE.md. Adapta la app para que funcione en el navegador local
   reemplazando window.storage, y verifica que todo siga funcionando."

## Contenido

- CLAUDE.md ............ Contexto completo del proyecto (léelo tú también)
- app/yeryscake-pos-v2.html ... La app actual completa
- assets/productos/ .... Las 87 fotos de productos en JPG individual

## Importante

La versión que está publicada/en uso en el iPad del local sigue funcionando
en Claude.ai de forma independiente. Este proyecto es para desarrollar la
versión comercial sin afectar la que usa la tienda.
